Fonctionnement du site

Manger au Foyer :

Etape 1 : Configurer ses amis

1) Cliquer dans le menu sur le bouton "amis".
2) Commencer a écrire le nom de votre ami dans la zone de texte ("Rechercher un ami").
3) Dès que vous voyez apparaitre sont nom dans la liste déroulante appuyez dessus puis cliquez sur le bouton "Ajouter".
4) Répéter les étapes 2 et 3 pour chacuns de vos amis.

Etape 2 : Inscription

1) Sélectionner la bonne semaine (la semaine actuelle correspond à la semaine encours).
2) Cliquer sur le créneau qui vous intéresse.
3) Sélectionner les amis avec lesquels vous souhaitez manger à cet horaire.
4) Votre compte sera débité du nombre de point nécessaire (ce nombre de point est par défaut 1 mais il peu y avoir des réductions qui seront indiqué sur la page d'accueil).
5) Vous pouvez alors valider en appuyant sur le boutton OUI.

Il sera toujours possible de modifier votre inscription par la suite.

ATTENTION , pour manger avec des amis, CHACUN de vos amis doit s'inscire au crénau voulu et mettre les sélectionnez les autres
Si l'un de vous n'es pas pris, tout le monde le sera aussi.
Si vous n'êtes pas pris vous ne perdez pas de point.

Etape 3 : Aller au foyer

1) Avant de vous rendre au foyer verifier si vous êtes inscrit pour cela regarder si le créneau demandé est devenu vert.
2) Si vous êtes accepté rendez-vous le jour J au foyer et presentez votre Pass.
2 bis) Sinon vous regagnez les points utilisés pour l'inscription.

ATTENTION , il est STRICTEMENT INTERDIT de falsifier son Pass et il est inutile de se presenter au foyer sans avoir été accepté.






Annexe :

Signaler sa presence en permance :

1) Rendez vous sur la page d'accueil et appuyé sur Planing Foyer(Permanence).
2) Cliquez sur semaine actuelle.
3) Cliquez sur le crénau voulu.
4) Assurez vous que personne dans votre classe n'a déja fait de demande dans votre groupe/classe.
5) Remplissez les champs et validez cela ne coute pas de points.

Retirer un amis :

1) Cliquer dans le menu sur le bouton amis.
2) Cliquez sur son nom dans la liste des amis pour le retirer.

Se désinscrire :

1) Cliquer dans le menu sur le crénau où vous êtes inscrit.
2) Appuyez sur OUI pour vous désinscrire.

Rajouter une personne avec qui manger :

1) Appuyez sur le créneau.
2) Apporter les modifications souhaités.

Enlever une personne avec qui manger :

1) Cliquer dans le menu sur le crénau où vous êtes inscrit.
2) Cliquez sur son nom dans la liste des amis avec qui vous voulez manger pour le retirer.
