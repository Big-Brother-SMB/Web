étape 1 : renseigner ses amis

1) Pour ce faire, appuyer dans le menu sur le bouton amis
2) Commencer a écrire le nom de votre ami dans la zone de texte (ami)
3) Dès que vous voyez apparaitre sont nom dans la liste déroulante appuyez dessus puis cliquez sur le bouton "Ajouter" Répeter les étapes 2 et 3 pour chacuns de vos amis
